# Renz Baloy — Portfolio

A simple, responsive portfolio website built with HTML and CSS. Deploy on GitHub Pages in minutes.

## Features
- Clean, accessible layout
- Mobile-responsive design
- Sections for About, Experience, Skills, Tools, Achievements, Contact

## Setup
1. Clone or download this repository.
2. Open `index.html` locally to preview.
3. Edit content in `index.html` as needed.
4. Commit and push to GitHub.

## Deploy to GitHub Pages
1. Go to your repository Settings.
2. Select Pages.
3. Set Branch: `main` and Folder: `/root`.
4. Save — your site will be live at `https://<username>.github.io/<repo>`.

## Customization
- Update colors and fonts in `styles.css`.
- Replace any placeholders in `assets/`.
